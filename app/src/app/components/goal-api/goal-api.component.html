Bienvenue sur Spotimatch !

Site web directement lié à notre Api et l'Api de Spotify. Notre but étant donnée le pourcentage de correspondance avec vous et vos amis !
Nous stocké en bd les inforamtions recues par l'Api de Spotify et analysons vos sons playlists et coup de coeurs en commun en fonction de ce que vous écoutez en ce moment !
Donc cela peut varié au fur et à messure le pourcentage n'est pas périn dans le temps.
Le stie web est très intuitif. Il suffit de vous connecter avec Spotify,
ce qui crééra par la suite un compte sur SpotiMatch.
Par la suite vous allez atterrir sur la Homepage.
Vous verez ensuite votre carte qui représente votre profil.
A partir de cet écran vous pouvez ajouter la carte d'un ami qui se trouve sur le site.
Pour ajouter un ami, il suffit de prendre son nom dans la liste d'ami et le drop dans la "drop zone"
Ensuite il suffit juste d'appuyer sur le bouton Match est le pourcentage s'affichera !
